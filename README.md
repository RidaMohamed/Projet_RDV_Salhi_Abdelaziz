# Projet_RDV_Salhi_Abdelaziz

## Compilation
```
Nous avons commencé le projet par faire le render d’une simple image comme début, Par la suite nous avons afficher la premier image out ‘ out_first.ppm’. Pour afficher le monstre mesh nous avonc utiliser la méthode orthographic projection.
Nous nous avons afficher le monstre avec algorithme of sweeping lines, ensuite avec le barycentric coordinates algorithm et avec la méthode de z-buffer et enfin avec la matrice de projection et le viewport.
Et pour finir nous avons fait la 3d.
```
```
git clone https://github.com/RidaMohamed/Projet_RDV_Salhi_Abdelaziz.git
cd Projet_RDV_Salhi_Abdelaziz
mkdir build
cd build
cmake ..
make
```
